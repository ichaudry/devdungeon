# DevDungeon

A roguelike dungeon crawler where programming concepts become weapons and software bugs become enemies.

## Tech Stack

- React
- TypeScript
- Vite
- Phaser 3 (coming soon)
- Zustand (coming soon)

## Development

```bash
npm install
npm run dev
```
